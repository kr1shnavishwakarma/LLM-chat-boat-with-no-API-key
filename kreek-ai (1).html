Settings

Kreek.ai needs an Anthropic API key to power responses. Get one at console.anthropic.com. It's stored only in this browser's local storage — never sent anywhere except directly to Anthropic's API.

⚠ For a real product, never ship an API key in client-side code that reaches end users — proxy requests through your own backend so the key stays secret.
